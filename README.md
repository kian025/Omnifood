# 🍽️ Omnifood

A modern and responsive landing page for a fictional food delivery service.

## ✨ Features

- 📱 Fully responsive design
- 🎨 Modern and clean UI
- ✨ Smooth scrolling navigation
- 📌 Sticky navigation
- 🎭 CSS animations
- 🚀 Optimized layout and typography

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript (ES6)
